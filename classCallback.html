Detailed Description

template<typename returnType, typename... argumentTypes>
class Callback< returnType, argumentTypes >

General purpose callback wrapper. Expects either c-style or c++ style function pointer, depending on if the C++ STL has been implemented on the given device or not.

Template Parameters
returnTypeType the given callback method should return
argumentTypesTypes the given callback method should receive
